성적 계산기 사용법과 성적 산출 방식

이 도구는 학교 성적이 결정되는 방식을 그대로 본떠, 시험 점수와 반영 비율만 입력하면 최종 점수·등급은 물론 “목표 등급에 필요한 점수”까지 계산해 줍니다. 아래 안내를 보면 처음 쓰는 분도 1분 안에 결과를 얻을 수 있습니다.

1. 사용 방법 (3단계)

  1. 반영 비율 설정 — 중간고사, 수행평가 2개, 기말고사가 최종 성적에서 차지하는 비율(%)을 입력합니다. 합계가 100%가 되도록 맞추면 가장 정확합니다.
  2. 점수 입력 — 각 평가의 획득 점수와 만점을 적습니다. 중간·기말고사는 객관식과 서술형을 따로 입력하거나, 한 번에 합산해 입력할 수 있습니다. 아직 보지 않은 시험은 비워 두세요.
  3. 결과 계산 — ‘결과 계산하기’ 버튼을 누르면 최종 점수, 등급, 그리고 차트가 나타납니다. PDF로 저장하거나 링크로 공유할 수도 있습니다.

2. 최종 점수는 어떻게 계산되나요?

각 평가의 득점률(획득 점수 ÷ 만점)에 그 평가의 반영 비율을 곱해 모두 더합니다. 예를 들어 다음과 같이 입력했다면,

  • 중간고사 90점 / 100점 (반영 30%) → 90% × 0.30 = 27점
  • 수행평가 1 : 18점 / 20점 (반영 20%) → 90% × 0.20 = 18점
  • 수행평가 2 : 16점 / 20점 (반영 20%) → 80% × 0.20 = 16점
  • 기말고사 85점 / 100점 (반영 30%) → 85% × 0.30 = 25.5점

최종 점수는 27 + 18 + 16 + 25.5 = 86.5점이 되고, 아래 등급 기준에 따라 B 등급이 됩니다.

3. 등급 기준

기본 등급 기준은 다음과 같습니다(필요에 맞게 코드에서 조정할 수 있습니다).

  • A : 90점 이상
  • B : 80점 이상 ~ 90점 미만
  • C : 70점 이상 ~ 80점 미만
  • D : 60점 이상 ~ 70점 미만
  • E : 60점 미만

4. “A를 받으려면 몇 점?” — 목표 점수 시뮬레이터

일부 시험만 보았고 나머지는 아직 남았다면, 미입력 항목을 비워 둔 채 계산해 보세요. 도구가 지금까지의 누적 점수와 달성 가능한 점수 범위를 보여 주고, 목표 등급(A·B·C·D)을 고르면 남은 시험에서 받아야 할 점수를 항목별로 알려 줍니다. 학습 계획을 세울 때 특히 유용합니다.

자주 묻는 질문 (FAQ)

입력한 점수가 서버에 저장되나요?
아니요. 모든 계산은 이용자의 브라우저 안에서 이루어지며 서버로 전송·저장되지 않습니다. 자세한 내용은 개인정보처리방침을 참고하세요.
객관식이나 서술형 중 하나만 입력해도 되나요?
네. 한쪽만 입력하면 입력한 영역을 기준으로 환산한 예상 점수가 표시됩니다.
반영 비율 합계가 100%가 아니어도 되나요?
계산은 되지만, 100점 만점 기준의 정확한 결과를 위해서는 합계를 100%로 맞추는 것을 권장합니다.
결과를 친구와 공유할 수 있나요?
‘성적표 공유’ 버튼을 누르면 입력 내용이 담긴 링크가 생성됩니다. PDF로 저장해 전달할 수도 있습니다.
이 도구는 무료인가요?
네, 완전히 무료입니다. 사이트 운영을 위해 일부 광고가 표시될 수 있습니다.
